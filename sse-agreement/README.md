# sse-agreement

> Server-Sent Events（通常简称为 SSE）是一种允许服务器向 Web 页面发送实时更新的技术。
> 
> 与 WebSocket 技术相比，SSE 专门设计用于从服务器到客户端的单向通信。
> 
> 这种单向性使其在某些场景中更为简单和直观。
